// IMPORT MEMBER LIST (TG DESKTOP)

Telegram Bot API does not officially expose getChatMembers. To see all ~32 actual group members, export the list manually:

1. Open Telegram Desktop → the group → ... → View Group Info → N Members.
2. Scroll the list to the end — Telegram will load everyone.
3. Copy the list (right-click → Copy in Telegram Desktop) or use «Export chat history» (Settings → Advanced → Export Telegram data → specific group, Members JSON only).
4. Paste here as a JSON array.

Supported format:
[{"telegram_id":118296372,"username":"tyoma_fyodorov","first_name":"Artyom"}]
Fields: telegram_id required; username, first_name, last_name, status, is_admin optional.
